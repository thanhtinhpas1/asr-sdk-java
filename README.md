# Speech to Text

## Installation

##### Maven

```xml
<dependency>
  <groupId>com.vietspeech</groupId>
  <artifactId>asr-vietspeech</artifactId>
  <version>0.0.1</version>
</dependency>
```

##### Gradle

```gradle
'com.vietspeech:asr-vietspeech:0.0.1'
```
